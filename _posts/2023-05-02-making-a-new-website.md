Recently, I got bored of the look of my website, so I decided to mix things up and create a new one. First, I was baffled by the vast number of softwares one can use to create a personal website. Initially, I thought that using plan <code>html</code> was the way, but soon I realized that I am not skilled enough to work with huge html files. Somehow I got to know <a href="www.google.chom"><code>jekyll</code> </a>, which is a static website builder tool. It turns out that quite a lot of people use it to create website. As a result you can find a large number of different templates that you can use as a starting point for your personal website. The template I use right now is called <a href="https://github.com/alshedivat/al-folio"><code>al-folio</code> </a>. It is a simple and elegant template which I like a lot to be honest. Also, decorating your website is easy due to the large number of tutorials that you can find on the web.

Needless to say that I should be writing my thesis :).
